Web ini dirancang untuk membantu Anda mengirim pesan WhatsApp ke nomor-nomor tertentu tanpa harus menyimpannya terlebih dahulu di kontak telepon.

Ya, pengiriman pesan melalui web ini tidak mengenakan biaya tambahan.

Masukkan nomor tujuan dengan kode negara (misalnya +62 untuk Indonesia) di kotak yang disediakan, lalu ketik pesan Anda dan klik tombol "Kirim". Pesan akan dikirimkan ke nomor WhatsApp yang ditentukan.

Tidak, pesan yang Anda kirimkan melalui web ini tidak akan disimpan di server kami. Setelah pesan dikirimkan, data pesan akan dihapus.
